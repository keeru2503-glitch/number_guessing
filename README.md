# Number Guessing Game

## Description

This is a command-line Number Guessing Game developed using Python. The player selects a difficulty level, and the computer generates a random number. The player continues guessing until the correct number is found.

## Features

- Three difficulty levels
  - Easy (1-10)
  - Medium (1-50)
  - Hard (1-100)
- Random number generation
- Input validation
- Unlimited guesses
- High/Low hints
- Attempt counter
- Play Again option

## Technologies Used

- Python 3
- Visual Studio Code
- Git
- GitHub

## How to Run

```bash
python guess_game.py
```
